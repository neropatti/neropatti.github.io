Undertale 10th anniversary echo flower timestamps

Did you whisper into an echo flower, and are now wondering if your message was visible on the anniversary streams?

I compiled a list of all names that appeared in the anniversary streams. You can search this page for the name you entered & get a timestamped link to your message being displayed!

I wrote some Rust code to automate most of this process, but in the end I did have to manually check every name to make sure they are correct. If you spot an error, please let me know! You can find me on discord, my handle is neropatti
